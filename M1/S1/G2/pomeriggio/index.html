Come creare un template per i post

di Mario Rossi - pubblicato il 25/10/2020

La prima cosa da fare è suddividere in blocchi logici la pagina, identificando le parti comuni da applicare a tutte le pagine

Le varie aree saranno quindi suddivise in:

  • contenuti di intestazione del tag header
  • contenuti centrali variabili nel tag main
  • contenuti di fine pagina comuni, da inserire ne tag footer

Formattazione del codice

Il testo dell' articolo dovrà essere formattato per identificare tutte le parti da identificare in CSS.
Perciò i passi saranno:

  1. progettare un markup assegnando ad ogni specifico testo il tag appropriato
  2. scrivere l' HTML dai tag esterni ai tag interni dall' alto verso il basso
  3. applicare gli stili di CSS

L' applicazione di stili CSS è cresciuta negli ultimi anni fino a raggiungere una diffusione pressocchè totale
